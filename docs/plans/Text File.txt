### 🟢 低风险 1：**`buildPropLevel` 中 `getPropList()` 调用可能返回空数组**

**代码**：
```typescript
const props = getPropList();
```
`getPropList()` 每次调用都返回新数组，在 `renderCustom` 中执行，无性能问题但浪费了一次数组分配。非紧急。

### 🟢 低风险 2：**`buildPropDetailLevel` 中多个相似滑块可提取为循环**

**问题**：
5 个 `addSliderRow` 调用结构几乎相同，只有参数不同，代码冗余。不影响功能，但降低可维护性。

**建议**：
提取为配置数组 + 循环调用。
