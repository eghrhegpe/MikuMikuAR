# MikuMikuAR v1.4.0

## 新功能

### 环境模块重构

- **env-water 水流模拟系统**
  - Babylon.js WaterMaterial 集成，CPU 法线纹理 + GPU 几何波动渲染
  - 48/16/6 三级 LOD 细分，距离阈值 30m / 80m 自适应切换
  - 平面反射（Planar Reflection）与环境贴图双路混合
- **env-lighting 光照系统**
  - 场景光照与环境预设（env_preset.go）完整接入 Go 后端
  - 灯光菜单声明式 Schema 重构，集成到统一环境面板
- **env-ground 地面系统**与 env-sky 天空分离
  - 独立地面模块（地面跟随逻辑属 ADR-085 部分实现）
  - 天空与地面分层渲染架构

### 场景环境菜单重构（ADR-111）

- 统一环境面板：地面 / 天空 / 云 / 水 / 光照 / 粒子归一化入口
- 声明式 Schema 驱动，57 面板全部迁移完成

### 道具库菜单拆分（ADR-111）

- 拆分为 actions（操作）/ browse（浏览）/ setup（设置）三个子面板
- 列表模式项实时重算，解决切换视角后快照陈旧问题

### 角色行三区交互

- 左区 Radio 切换焦点角色
- 中区文本点击进入详情
- 右区齿轮打开工具菜单

### 状态栏与快捷键

- 完整状态栏 UI（status-bar 模块）
- 快捷键系统（shortcut-app）
- 应用级事件总线（events）

### 动作系统修复

- **修复缩略图截帧时动画被清空**：vmd-loader 的 `stopProcMotion` 延迟到 `vmdData` 赋值之后
- **修复动作叠加多余层**：添加动作时无条件替换基础动作
- **移除动作弹窗「最近使用」死代码**

### 国际化补全

- 5 个 locale 补全 10 个缺失翻译 key
