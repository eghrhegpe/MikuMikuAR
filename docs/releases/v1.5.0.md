# MikuMikuAR v1.5.0

## 新功能

### 渲染与视觉

- **ADR-114 Phase 2+3：地面反射增强** — 反射模糊（mipmap+roughness）、法线扭曲（bumpTexture）支持；接触阴影通过屏幕空间 ray marching 后处理实现，中/高质量守卫
- **ADR-113 Phase C/D：体积云效果增强** — 背光輝光（backlight）+ 粉末效果（powder），云层渲染质量提升
- **缩略图系统全面治理**：
  - 缓存键单一化（`thumbnail-key.ts` 纯函数 + 契约测试）
  - 舞台缩略图 16:9 横屏 / 角色 2:3 竖屏比例区分
  - 冻结缩略图相机投影矩阵与物理，修复 RTT 比例压扁与物理模糊
  - 高 DPI 适配（纳入 `devicePixelRatio` + `GL_MAX_TEXTURE_SIZE` 钳位）
  - 缩略图卡片尺寸 80px→100px

### 动作系统

- **ADR-116（已完成）：动作覆盖系统重构** — Bone Override → Motion Override 全面重写
  - 引擎位置覆盖扩展 + 持久化
  - 4 个语义模块（position-offset + 右手独立控制）
  - `claimBones` 按 priority 抢占仲裁（F1）
  - 审核报告 P1-P4 全部修复
- **ADR-121：全局动作意图（Phase 1）** — Scene-level Motion Intent 初步落地
- **ADR-118：刷新率感知自动降级** — 降级阈值改为刷新率相对标定，120Hz 适配，60Hz 零回归

### 环境系统

- **ADR-120 Phase 1：环境预设分类化** — 天空/地面/水面/大气预设分类
- **环境模块重构** — env-clouds/ground/sky/water 各子模块拆分重构，env-bridge 同步
- **水面持久化修复** — `reflectionQuality` 写入 Go 端，泡沫/水下参数实时同步
- **平面反射引擎重构** — 统一贴图工厂，修复反射双向失效

### 感知层交互

- **情绪菜单** — 情绪按钮点击 + 微表情/表情合并为 folder，情绪选项展开显示
- **眨眼幅度参数**（`blinkAmplitude`）对外暴露
- **重心微动**迁回程序化 idle VMD 生成
- 口型同步 UI 去重 — 移除程序化动作菜单中的重复项
- 感知层滑块功能 — 将硬编码参数暴露给用户调整

### 音效系统

- **脚步声检测（footstep-detect-fallback）** — audio-bus 集成 + footstep 模块
- 道具/换装音频槽位同步

### 后端与基础设施

- **ADR-117 Phase 1+2：Go 端错误 i18n 化** — `i18nerr` 包、goerr 桥接、integration.go、bone-override 整合
- **MPR 构建基础设施** — build-tagged files、zipextract、coep 测试
- **替换模型 ZIP 缩略图修复** — replace 路径补传 `innerPath`

### 其他改进

- 全部 console.error 收敛为 `logError`
- 提取预设芯片组与清除行通用 helper
- 感知层菜单开关合并至 folder headerToggle
- `build/windows/info.json` 版本同步（1.0.0 → 1.5.0）
- 文档整顿 — `docs/` 目录职责分配清理
