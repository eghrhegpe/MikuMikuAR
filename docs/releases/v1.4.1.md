# MikuMikuAR v1.4.1

## Bug 修复

- **修复 ZIP 模型缩略图永不命中**（`frontend/src/menus/library-core.ts`）
  - `thumbnailKeyForModel()` 原直接返回 `filePath`，与 `model-loader.ts captureThumbnail()` 保存时使用的 key 格式不一致（ZIP 内模型应为 `filePath::zipInner`）。
  - 现已对齐：ZIP 内模型返回 `` `${fp}::${m.zip_inner}` ``，普通模型返回 `fp`。
  - 修复后 GetThumbnailBatch 查询可正确命中 ZIP 内模型的截图缓存。

---

> 详情见 `frontend/src/menus/library-core.ts:thumbnailKeyForModel()`。
