# MikuMikuAR v1.5.2

## 新功能

### 动作系统全面重构

- **ADR-129 Phase 1-3：场景级动作 UI 重构** — 双槽位动作系统、场景级动作菜单重写、模型详情卡片化、动作命名链路对齐修复（motion-popup 入口卡片与场景面板 i18n key 错位）
- **ADR-121 P1：全局动作意图** — motion-intent 单例（get/setActiveMotion + 广播回调），替换 pendingVmd；场景文件持久化；两级兼容性匹配 + 广播加载去重
- **ADR-125 P2：撤销/重做** — 模块层历史栈（per-model 隔离 + 时间窗口合并）、Ctrl+Z/Ctrl+Shift+Z 快捷键、撤销/重做/历史下拉按钮 UI
- **ADR-126 Phase 1-3：变换适配器** — 跨 kind 变换适配器注册表去重、拖拽/数值双模态实时同步、网格吸附（Gizmo snapDistance）
- **ADR-127：场景级破坏性操作撤销** — Memento 快照 + 撤销 toast，info 变体双色边框
- **ADR-116：动作覆盖模块增强** — 手部位置偏移 + 默认值回退

### 场景与环境

- **ADR-111：场景菜单重组** — 地面/水面/镜面迁入场景文件夹路由；env-menu 移除 env:lighting/ground/water 路由，改走 slideRow headerToggle + 箭头导航全参数
- **ADR-132：环境亮度统一** — 体积云响应 skyBrightness，云随天空亮度/色调联动
- **ADR-115：水面/体积云调优** — 移除泡沫参数、CLOUD_LIGHT_STEPS 自阴影落地、渲染质量调优
- **ADR-086：地面碰撞特性** — physics 地面碰撞落地

### Android 与构建

- **ADR-133：Android MPR 构建门控** — MPR 条件编译 + 广场 Android 处理

### 资源管理与 E2E

- **ADR-131：资源浏览选中后行为统一契约** — BrowseOutcome
- E2E 大幅增强：DOM 契约迁移（actor:model）、场景/动作/云/水 e2e、后处理断言重定位、契约对齐

### 其他改进

- 呼吸轴向修正 Y→X（ADR-071）+ 回归测试
- AutoDance 程序化算法细化：肘部骨骼、节拍栅格、无缝循环（ADR-021）
- 感知层程序化动作 per-slot 修复，不被 updateProcMotion 误停
- 虚拟裙骨构建失败错误上报五语种
- EnvState 契约对齐去泡沫 + 补字段
- 环境 time-of-day 双源拆分（持久源+暂停标志）
- HMR disposeScene 级联释放 + 各子系统 isDisposed 防御
- 全局 prettier 格式化（frontend/src 56 文件）
- .gitattributes catch-all eol=lf 根治行尾噪声
- Ctrl+5 快捷键切换设置面板
- 动作变更触发自动保存
