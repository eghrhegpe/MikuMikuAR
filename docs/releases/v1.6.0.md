# MikuMikuAR v1.6.0

## 新功能

### 感知层系统（ADR-160~166）

- **per-model 感知实例化** — 每个模型独立感知上下文，消除跨模型状态污染
- **冲突可视化** — claimBones + banner + 滑块标记，直观展示骨骼占用冲突
- **全员感知 + 自动性能降级** — Phase 2 落地，根据模型数量自动调整感知精度
- **per-context 对象池** — 替代全局单例，消除多模型覆写污染
- **gaze delta 指数衰减** — 替换固定 Slerp 为 dt 驱动，注视更自然

### 动态追光系统（ADR-168/169）

- **FollowTarget 类型 + tickFollowLights 逐帧追踪** — 骨骼缓存 + UI 跟随目标卡片 + 5 语言 i18n
- **动作装载语义统一** — `replaceDefaultMotion` 原位替换默认动作

### 体积光 / 光锥（ADR-152）

- **真实光锥 mesh + ShaderMaterial** — 替代假 VolumetricLightScatteringPostProcess，支持 autoSave + 旧存档迁移

### 反射系统统一架构（ADR-151）

- **平面反射 + 反射模式菜单入口** — reflectionQuality 与 reflectionMode 拆分，qualityProfile 派生
- **balanceSway 独立参数暴露** — 周期 + 振幅可调

### 雨天效果

- **地面涟漪系统** — slot 数可配置（16-1024），shader rings=1 环 + 显式始末半径插值
- **湿身效果** — 雨天角色材质湿润感

### 场景交互

- **场景级拖拽模式（ADR-171）** — 快捷开关 + 收纳文件夹
- **动作库选中范式（ADR-170）** — 复用默认指针为行级选中，删除/收藏收入工具页
- **场景级撤销** — Ctrl+Z 回退场景操作

### 动作系统

- **MotionPipeline 显式调度器（ADR-147）** — 帧钩子显式 order 排序，根治双观察者隐式定序
- **BoneOverrideStore 集中状态** — IK 感知骨骼覆盖 + absolute 模式（ADR-122/123）
- **动作预设 i18n + UI 完善（ADR-145）**

### 环境与渲染

- **无限地面（ADR-134）** — smooth-lerp 相机追踪跟随，世界空间 UV 补偿
- **程序化地面纹理** — wood/marble/concrete + 法线扭曲增强
- **质量配置（ADR-130）** — qualityProfile + 性能模式 + env-bridge 集成
- **刷新率感知自动降级（ADR-118）** — 运行时峰值校准全量落地
- **程序化天空 IBL** — PBR 材质环境立方体贴图，修复镜面预设纯黑

### Android

- **屏幕方向切换（ADR-017）** — 竖/横/自动
- **屏幕常亮 + WebView 渲染崩溃自愈**
- **返回键双击退出**

### AR 探针（ADR-072/073）

- **WebXR/ARCore/Vuforia 可行性验证** — 非侵入式深度探针 + Android Java 侧协同

### 无障碍（ADR-153）

- **三阶段路线图立项** — 基础键盘导航 → 屏幕阅读器 → 高级适配
