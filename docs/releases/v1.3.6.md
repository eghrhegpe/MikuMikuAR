# MikuMikuAR v1.3.6

## 构建系统修复（无用户功能变更）

- **Windows 构建脚本 UTF-8 编码健壮性**（`scripts/build-windows.ps1`）
  - 补 UTF-8 BOM，修复本地 `powershell`(PS5.1, ANSI 代码页) 跑 `npm run build:win` 时整脚本 `ParserError: 字符串缺少终止符`（中文被当 Latin-1 解码致引号错位）的解析失败。
  - 版本注入的 `Get-Content`/`Set-Content` 显式指定 `-Encoding UTF8`，修复每次发版把 `build/config.yml`、`build/windows/Taskfile.yml` 的中文注释/字符串从 UTF-8 重编码为 GBK 损坏（CI 用 `pwsh`(PS7) 默认 UTF-8 不踩，但本地 PS5.1 必踩）的问题。
- `scripts/setup-github-secrets.ps1` 同步补 UTF-8 BOM。
- 配套文档：新增 `docs/releases/release-process.md` 发版标准作业程序 SOP（人类操作视角）。

> 本次为纯 CI/构建链路改进，应用运行时行为与 v1.3.5 一致。
