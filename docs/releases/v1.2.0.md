# MikuMikuAR v1.2.0 Release Notes

## 🎯 核心亮点

- **i18n 国际化框架全面落地（Phase 1–4）**：全运行时 UI 字符串 `t()` 化，新增日语 / 韩语 / 繁体中文语言包，运行时热切换
- **Pose Studio 姿态工作室**：姿态预设 + 构图辅助线 + 相机角度预设 + Wails v3 源码分析
- **AR 相机模式**：现实场景叠加，配合异步竞态修复
- **WASM Motion Layers 混合器**：动作分层混合，E2E 测试框架同步落地
- **全屏资源库面板**：资源浏览体验升级 + Motion Override + Accessory 道具骨骼挂载（卸载自动 detach）
- **水面高级参数系统**：颜色感知焦散纹理 + 高级参数 UI
- **高度图地形系统**：`TerrainHeight` 语义明确化 + `getGroundHeightAt` 加固 + 单元测试（ADR-061 骨骼控制规划）
- **设置模块化重构**：settings / procedural-motion 抽取独立模块
- **自动更新开关**：落地运行时开关
- **安全审计**：CVE 审计，加固 vpd-parser / zipextract / audio / beat / motion，下架实验性 store

## ✨ 新功能

### 国际化与本地化
- **i18n 框架 Phase 1–4**：菜单与对话框 `t()` 化收尾，新增 ja/ko/zh-TW locales，中英语言包同步扩充

### 动作与动画
- **Pose Studio**：姿态预设生成 + 构图辅助线 + 相机角度预设
- **WASM Motion Layers**：分层混合器，支持动作叠加与过渡
- **Gaze layer**：头部 / 眼部追踪桥接 + VMD 骨骼过滤

### 菜单与 UI
- **全屏资源库面板**：资源浏览体验升级
- **Motion Override + Accessory**：动作覆盖与道具骨骼挂载（挂载自动 detach）
- **outfit-overlay 换装叠加层**：换装系统新模块
- **状态栏 / Toast 组件**：统一错误提示（`tryCatchStatus` 扩展）
- **相机反 Y 轴支持**：设置项 + 类型同步

### 环境与渲染
- **水面高级参数**：颜色感知焦散纹理 + 高级参数 UI
- **高度图地形**：`TerrainHeight` 语义明确化 + `getGroundHeightAt` 加固

### 其他
- **自动更新开关**：落地运行时开关
- **AR 相机模式**：现实场景叠加

## 🔧 改进

- **设置模块模块化**：settings / procedural-motion 抽取独立模块，Android 构建资源与 i18n 同步
- **库扫描并行化**：`SafeCall` 统一 + 清理冗余扫描代码
- **预设动画天空纹理重建节流**：每 3 帧一次
- **模型详情页重构**：`基本信息` 标签 + 材质按部位筛选
- **文档瘦身**：删除过期文档，menu-architecture 重命名
- **E2E 测试框架**：固化 `MMCAR_DEBUG_PORT` 注入机制，env-sky / @webgl 规格落地
- **CI 自动检查**：恢复手写 bindings + 回写 status 事实
- **统一错误处理**：toast + `tryCatchStatus`

## 🐛 修复

- **AR 相机异步竞态**：修复 `scene` 状态不同步与初始化竞争
- **水面 LOD 重叠 + 波相位跳变**：修复预设持久化问题，补充单元测试
- **go embed 失败**：test-backend 需要 frontend dist 的构建顺序修复
- **i18n 键对齐**：`env.envLighting` 与 sky-lighting 标签对齐；天空纹理提示消除歧义
- **模型信息材质数修正**
- **playback 栏占位 UI**：无 mmdRuntime 时显示占位
- **nav 按钮注册提前**：修复 `initScene` 之前注册时机

## 📦 构建

- **bindings 迁移至 JS**：测试债清理 + 自动更新落地
- **WASM Motion Layers**：混合器 + E2E 测试
- **Windows / Android / Linux**：三平台构建
