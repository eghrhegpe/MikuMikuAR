# MikuMikuAR v1.3.1

**发布日期**：2026-07-13
**类型**：功能更新版

## 本版亮点

- **ADR-093 声明式菜单 Schema 全量落地**：全部 55 个面板（天空/地面/水面/风/云/阴影/雾/实验/设置/运动/场景/灯光等）从命令式 UI Builder 迁移至 `MenuNode` + `renderMenu()` 单渲染器架构，StatePath 前缀 + ControlSpec 衍生控件 + visibleWhen 守卫，菜单结构从此可追踪、可测试。
- **文件对话框记忆上次目录（ADR-090）**：模型导入 / 动作加载 / 贴图选择等文件对话框按类型分别记忆最近访问目录，跨启动保留，显著减少路径切换成本。
- **环境系统增强**：环境功能层级重构，粒子/云层/水面渲染管线完善，ghost-particle-sim 遗留实现移除。
- **库管理增强**：资源库面板交互优化，模型预设与缩略图生成逻辑重构。

## 已知限制

| 限制 | 说明 |
|------|------|
| Windows 端目录选择器 | `SelectDir` 仍受 Wails v3 `CanChooseDirectories` 缺陷影响，弹出文件选择器而非目录选择器（待修复，见 ADR-023 §4） |
| Android localStorage 容量 | 场景自动保存写 localStorage，Android 有 5MB 限制，大场景可能写满 |
| AudioContext 惰性创建 | Android WebView 需用户交互后才能创建 AudioContext，首次无声音频可能失败 |

## 更新方式

**Windows**：下载 `MikuMikuAR-1.3.1-windows-amd64.exe`，覆盖安装即可。

**Linux**：下载 `mikumikuar-1.3.1-linux-amd64`，赋予执行权限后运行。

**Android**：下载 `MikuMikuAR-1.3.1-android-arm64.apk` 安装（Release 签名版本）。

## 升级注意

- 配置文件（`.mmascene` / 用户设置）向后兼容，无需迁移。
- 若出现异常，可尝试 `Ctrl+Shift+R` 重置本地缓存后重启。
