# MikuMikuAR v1.5.1

## 新功能

### 风格化水面波光（ADR-115 全阶段完成）

- **P1：焦散强度 UI 暴露** — 水面焦散效果现可通过滑块调节
- **P2：高频法线扰动层 + Sun Glitter** — 双层法线扰动增强水面细节，太阳反光闪烁效果
- **P3：地平线淡出 + 天空-水面颜色联动** — 远距离水面自然融入天空，颜色随环境自动匹配
- **P4：双尺度波高** — 大/小波独立振幅控制，波浪细节更丰富

### 骨骼与动作系统

- **ADR-122：ik-aware-bone-override** — IK 感知骨骼覆盖，避免覆盖与 IK 链冲突
- **ADR-123：compute-override-semantics** — 计算覆盖语义规范，明确覆盖优先级与合并规则

### 文件系统与构建

- **ADR-124 Phase 2：fileaccess 迁移修尾** — `readFileBytes` 返回 `Uint8Array` 适配，模型纹理递归直传 `referenceFiles`
- **ADR-117 Phase 3：goerr-lint CI 门禁** — 新增 CI 检查 `scripts/goerr-lint.mjs --strict`，禁止裸中文 `fmt.Errorf` 与前端直显反模式，新增用户可见错误必须走 i18n 化路径

### UI 与体验

- 菜单空状态引导提示（无表情/无舞台加载时的指引文案）
- 复用 `addToggleRow`/`addModeRow` 统一 stage/prop/morph/camera 骨骼选择面板
- 环境粒子 level builder 与纹理选择器统一

### 文档

- 发版 SOP 完善（`docs/releases/release-process.md`）：步骤从 7 步扩为 9 步，新增常见坑 §10、快速命令序列 §11
- 小说新增两章：《水面苏醒》《HTTP 的黄昏》
