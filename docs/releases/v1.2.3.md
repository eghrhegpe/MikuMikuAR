# MikuMikuAR v1.2.3 Release Notes

## 🎯 核心亮点

- **版本号注入修复**：Windows + Android 发布构建现在通过 `-ldflags` 正确注入 `AppVersion`，修复关于 / 检查更新页显示 `dev` 的遗留问题
- **AR 接触阴影**：用 blob-shadow projector 思路替代悬浮飘感，模型落地更稳
- **图标离线固化**：图标资源固化进 bundle，运行期零网络回退，裁剪 SEED 死重
- **相机环绕模式拆分**：surround 从 concert（fan-cam）语义独立（ADR-070）
- **Android AR 相机集成**：原生 AR 相机接入 + 程序化动作打磨
- **材质纹理槽支持 + 相机骨骼锁定 UI**：材质系统增强
- **程序化动作 / 感知边界拆分**：感知逻辑从 `proc-motion-bridge` 抽出（ADR-071）

## ✨ 新功能

### AR
- **接触阴影**：blob-shadow projector 替代悬浮感
- **Android AR 相机集成**：原生 AR 相机接入

### 相机
- **surround 模式拆分（ADR-070）**：与 concert fan-cam 语义解耦

### 材质
- **纹理槽支持**：材质纹理槽 + 相机骨骼锁定 UI
- **i18n / 库菜单同步**

### 工程
- **离线固化图标**：进 bundle，零网络回退
- **程序化动作 vs 感知边界（ADR-071）**

## 🔧 改进

- **AppVersion 注入**：Windows（`build-windows.ps1`）/ Android（`build-android-so.ps1`）发布构建补齐 `-ldflags`
- **perception 逻辑拆分**：出 `proc-motion-bridge`，职责清晰

## 🐛 修复

- **ESLint 阻断 CI**：修复 `no-func-assign` / `no-unsafe-finally`
- **model-detail-ui 断言同步**：可见性 → 透明度（AI 菜单收纳）
- **AppVersion 显示 dev**：注入修复（见亮点）

## 📦 构建

- Windows / Android / Linux 三平台，版本号正确注入

## ⬆️ 升级说明

覆盖安装即可。建议从 v1.2.0 / v1.2.1 升级以修复版本号显示问题。
