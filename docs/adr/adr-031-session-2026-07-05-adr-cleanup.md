## 教训

- 文档厚度与 AI 犯错概率正相关
- 规划文本是给人类看的，AI 需要的是决策记录
- 一份路由表 + 一份架构 + 一份 UI 规范 = AI 能干活的最小集
- 三份索引等于没有索引——坚持单一路由表原则
