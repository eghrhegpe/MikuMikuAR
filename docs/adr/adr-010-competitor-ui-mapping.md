# ADR-010: 竞品功能 UI 映射参考

**日期**：2026-07-16  
**来源**：lio.cc.cd MMD Viewer 功能分析
> **状态**: 参考文档 — 迭代参考路线图

---

竞品 lio.cc.cd 是纯浏览器 MMD 播放器，渲染层面功能丰富。本文档将其功能按 MikuMikuAR 的现有 UI 架构做归类映射，供后续迭代参考。

## UI 架构回顾

```
┌─────────────────────────────────┐
│  模型库弹窗 (library.ts)        │ ← 🎵 按钮
│  ├── 加载模型                   │
│  ├── 场景 (动态)                │
│  ├── 相机模式                    │
│  ├── 动作倍率                    │
│  ├── 角色定制                    │
│  └── 重新扫描                   │
├─────────────────────────────────┤
│  动作弹窗                        │ ← 💃 按钮
│  └── VMD 列表 + 播放控制         │
├─────────────────────────────────┤
│  场景弹窗 (scene-menu.ts)        │ ← 🎬 按钮
│  ├── 相机模式切换                │
│  └── 灯光控制                    │
├─────────────────────────────────┤
│  设置弹窗 (settings.ts)          │ ← ⚙ 按钮
│  ├── 显示名优先级                 │
│  ├── 外部库管理                  │
│  └── 下载监听                    │
├─────────────────────────────────┤
│  模型详情面板 (adr-009)          │ ← 点击已加载模型
│  └── 模型信息 + 操作             │
└─────────────────────────────────┘
```

## 竞品功能 → 映射

### 🎬 场景弹窗 (`scene-menu.ts`) — 当前只有相机+灯光

| 竞品功能 | 当前状态 | 映射到 |
|---------|---------|--------|
| 后处理滤镜 (Bloom/DOF/SSS/暗角/锐化/FXAA) | ❌ | → 场景弹窗新增「渲染」子菜单 |
| 色调映射 (ACES/Reinhard/Cineon 等 6 种) | ❌ | → 同上「渲染」子菜单 |
| 舞台效果 (反射地面/发光环/网格) | ❌ | → 场景弹窗新增「舞台」子菜单 |
| 阴影质量 (分辨率/模糊) | ❌ | → 同上「舞台」子菜单 |
| 环境 (反射强度/雾效/背景色) | ❌ | → 同上「舞台」子菜单 |
| 预设系统 (10+ 风格一键切换) | ❌ | → 场景弹窗新增「预设」子菜单 |
| 导入/导出渲染参数 (JSON) | ❌ | → 场景弹窗底部 action 行 |
| 动画速度 (0.1x~3x) | ❌ | → 已在模型弹窗「动作倍率」✅ |
| 重力 | ❌ | → 场景弹窗「物理」子菜单（如加） |
| 相机 FOV (20°~120°) | ❌ | → 场景弹窗「相机」子菜单 |

### 🎨 模型详情面板 (`model-details`，adr-009)

| 竞品功能 | 当前状态 | 映射到 |
|---------|---------|--------|
| 材质编辑 (皮肤/头发/眼睛/衣服分部位) | ❌ | → 模型详情面板「材质」tab |
| 单独材质编辑器 (每个材质独立调参) | ❌ | → 同上，材质列表可展开 |
| 材质高亮定位 (其他变半透明) | ❌ | → 材质列表行 click → Babylon highlight |

### ⚙ 设置弹窗 (`settings.ts`)

| 竞品功能 | 映射到 |
|---------|--------|
| 中英文切换 | → 设置弹窗「语言」项 |
| 移动端适配 (响应式) | → 远期，非 Phase 1-2 |
| 导入/导出整体配置 | → 设置弹窗底部 action 行 |

### 🎵 模型库弹窗 (`library.ts`)

| 竞品功能 | 映射到 |
|---------|--------|
| 音乐加载 (MP3/WAV/OGG) | → 不在路线图中（专注 PMX/VMD） |
| 内置资源库 | → 已有模型库弹窗 ✅ |

## 实施优先级

| Phase | 功能集群 | 复杂度 | 竞品对标度 |
|-------|---------|--------|-----------|
| **Phase 3** | 后处理滤镜 + 色调映射 | 中 | ⭐⭐⭐ |
| **Phase 3** | 舞台效果 + 阴影 + 环境 | 中 | ⭐⭐⭐ |
| **Phase 4** | 材质编辑器 (分部位) | 高 | ⭐⭐⭐⭐⭐ |
| **Phase 4** | 预设系统 | 中 | ⭐⭐⭐ |
| **Phase 5** | 中英文 + i18n | 低 | ⭐ |

> Phase 1-2 不变：标签系统 + 下载目录监听。
