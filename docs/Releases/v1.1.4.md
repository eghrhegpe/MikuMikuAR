# MikuMikuAR v1.1.4 Release Notes

## 🎯 核心亮点

- **菜单系统统一化**：完成 MenuStack 架构升级，场景/环境菜单完全解耦，新增 `envStack` 独立实现
- **VMD 分层系统**：新增 `vmd-layers.ts`，支持动作分层叠加与过渡混合
- **设置面板重构**：`settings-targets.ts` 独立模块，UI 组件统一化（slideRow/addCollapsible 等拆分为独立文件）
- **CI 全绿**：修复 Go 测试 `TestManifestJSON` 常量断言、Wails 绑定 `Create` mock、PowerShell Core keystore 解码等问题

## ✨ 新功能

### 动作与动画
- **VMD 分层系统**（`vmd-layers.ts`）：支持多动作分层叠加、过渡混合、优先级管理
- **程序化动作桥接层**（`proc-motion-bridge.ts`）：统一程序化舞蹈与 VMD 的播放接口
- **LipSync 桥接层**（`lipsync-bridge.ts`）：音频驱动口型同步，支持实时麦克风输入

### 菜单与 UI
- **环境菜单升格**：新建 `envStack` 变量，`env-menu.ts` 升格为独立一级菜单，与场景菜单完全解耦
- **设置面板重构**：`settings-targets.ts` 独立模块，统一管理设置目标（快捷键/渲染/音频等）
- **UI 组件拆分**：`slideRow` / `addCollapsible` / `addModeSlider` 从 `ui-helpers.ts` 拆分为独立模块

### 物理与布料
- **XPBD 布料管理器**：`cloth-manager.ts` 接入主场景，支持裙摆物理参数 UI 调节
- **物理菜单拆分**：物理设置从场景菜单独立，新增刚体/布料/碰撞体三级面板

## 🔧 改进

- **帧率限制**：设置页新增帧率滑条（0=不限 / 30 / 60 / 144），渲染循环节流
- **预设动画天空纹理重建节流**：每 3 帧一次，CPU 占用从 ~120 次/秒降至 ~40 次/秒
- **模型详情页重构**：`模型信息` 标签改为 `基本信息`，材质调节面板支持按部位筛选
- **文档瘦身**：删除 8 个过期文档（`foundation.md` / `requirements.md` 等），`menu-architecture.md` 重命名为 `menu-how-to.md`
- **ADR 体系完善**：新增 ADR-031（session cleanup）/ ADR-032（config-split）/ ADR-033（菜单统一化）/ ADR-034（设置差距分析）/ ADR-035（快捷键注册表）

## 🐛 修复

- **CI 测试全绿**（30 套件 / 846 测试用例）
  - 修复 `@wailsio/runtime` mock 缺失 `Create` 导出（13 套件恢复）
  - 修复 `model-detail-ui.test.ts` 缺失 `../scene/scene` mock（12 用例恢复）
  - 修复 Go `TestManifestJSON` 硬编码版本号 5 → 动态引用 `extractCacheVersion` 常量
  - 修复 PowerShell Core `Set-Content -Encoding Byte` 废弃问题，改用 `WriteAllBytes`
- **构建脚本降级**：`wails3 build` 失败时自动降级为 `go build`，避免 alpha 版 bug 阻塞构建
- **Playback dispose guard**：`_disposed` 防重复清理 + `async seekAnimation` catch 处理 + loop 边界检查
- **WASM 降级提示**：Bullet 物理初始化失败时给出明确提示，而非静默失败
- **weight 归一化**：MMD 骨骼权重归一化处理，修复部分模型形变异常

## 📦 构建

- **Windows**：`MikuMikuAR.exe`（~27 MB），支持 Windows 10+
- **Android**：实验性 APK（arm64），c-shared + WebView2
- **Linux**：AMD64 二进制（未实测）
