# MikuMikuAR v1.2.6

> 构建/CI 质量版本，无用户可见功能变更。

## 改动

- **CI 依赖缓存修复（release.yml）**：`npm ci` 改为受 `node_modules` 缓存命中守卫限制。
  此前 `build-windows.ps1` 无条件执行 `npm ci`，其首步即删除 `node_modules` 重建，
  使上游缓存形同虚设。修复后命中缓存即跳过安装，显著缩短前端依赖准备耗时。
- **wails3 版本同源（build.ps1）**：本地依赖检查脚本的 wails3 安装提示由写死 `@latest`
  改为从 `go.mod` 动态读取 `wails/v3` 版本，与 `release.yml` 的 `prepare.wails_ver` 同源，
  杜绝 CLI/库版本漂移。
- **wails3 二进制缓存**：Windows / Linux / Android 三平台均按 `wails-<版本>-<OS>` 缓存
  `wails3` CLI，仅在缓存未命中时 `go install`，避免每次重复编译（约 1 分钟）。

## 验证

- 本版本主要验证 GitHub Actions 的 `wails3` 与 `node_modules` 缓存命中行为。
- 应用产物与 v1.2.5 功能一致（地面范围 / 边缘淡出等）。
