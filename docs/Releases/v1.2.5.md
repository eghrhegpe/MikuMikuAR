# MikuMikuAR v1.2.5 Release Notes

## 🎯 核心亮点

- **感知层（Perception）**：微表情 / 躯干微晃 / 口型同步 实时叠加，迁入独立感知模块（ADR-079）
- **水面平面反射系统**：ADR-062 P1+P2 水面反射
- **预设系统重构**：灯光 indicatorScale / 透明度 + DOM 清理 + 死代码移除
- **地面范围 + 边缘淡出**：类比水面 `waterSize` 的地面范围设置 + 径向 opacity 边缘淡出
- **模型广场增强**：Cookie 中继 + 下载拦截一键入库 + 页面语义提取（ADR-077/078/079）
- **AR 平面检测 / 原生 ARCore-ARKit**：+ 赛璐珞后处理
- **全局状态层抽出**：menu-factory 注册收敛 + 样式 / 测试同步
- **纯 items 层级语言热刷新（ADR-065）**
- **Android branded launcher icons**：从 appicon master 渲染品牌启动图标

## ✨ 新功能

### 动作与感知
- **感知层**：微表情 morph / balanceSway / lip-sync 实时叠加（ADR-079）

### 环境与渲染
- **水面平面反射（ADR-062 P1+P2）**
- **地面范围 + 边缘淡出**：径向 opacity 贴图

### 菜单与 UI
- **motion-gaze-levels**：lip-sync 开关与滑块、微表情开关与情绪选择
- **全局状态层**：菜单注册收敛

### AR 与广场
- **AR 平面检测 + 原生 ARCore-ARKit**：+ 赛璐珞后处理
- **模型广场**：Cookie 中继 + 下载拦截入库

## 🔧 改进

- **预设系统重构**：灯光 indicatorScale / 透明度 + DOM 清理 + 死代码移除
- **全局状态层抽出**：menu-factory 注册收敛 + 样式 / 测试同步
- **渲染循环 + 环境影响计时埋点**：修复降级反馈循环
- **全量 prettier / eslint --fix 格式化**：修复 ESM sourceType 误判、ignore POC
- **Android branded launcher icons**：从 appicon 渲染
- **纯 items 层级语言热刷新（ADR-065）**

## 🐛 修复

- **降级反馈循环**：渲染循环与环境影响计时修复
- **lint 修复**：`plaza.ts` iframe.src 自赋值、`vite-env.d.ts` ImportMeta 移入 declare global 等
- **ADR 断链**：adr-079 → adr-016 链接修复

## 📦 构建

- **移除 TS XPBD 实验性物理全栈**：开发期内曾尝试基于 TS XPBD 的 Ragdoll 布娃娃（参数 / 预设 / blendWeight 平滑过渡 / 球体约束可视化调试）与可挂载布料（塌地修复、自动适配原生坐标缩放、多卡片增量刷新），因巨量 bug 回滚；**v1.2.5 仅保留 WASM MMD 原生物理、风力与物理分类开关**（ADR-019 / ADR-061-r 相关的实验特性收尾，无 WASM 等价物随之消失）
- Windows / Android / Linux 三平台
