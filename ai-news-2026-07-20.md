# AI 领域每日要闻 · 2026-07-20

> 侧重 AI coding 与具身智能两条主线，共 5 条。当日为 WAIC 2026（7/17–20 上海）收官日，具身智能主线以"从身体竞赛到大脑竞赛"为总基调。

## 一、AI Coding

### 1. Meta 发布 Muse Spark 1.1，正式进军 agentic coding
Zuckerberg 三年来首次在 X 发帖为其站台。Muse Spark 1.1 是 Meta 超级智能实验室（MSL）训练的多模态推理模型，主打 agentic coding 与智能体任务：可修复代码漏洞、协助大型代码迁移，并在跨多外部应用/服务的长时间会话中保持上下文、在脚本/直接界面交互/批量操作间智能切换；在部分第三方编程产品交互任务中表现据称超过竞品。
- **值得关注**：继 OpenAI、Anthropic 之后，Meta 把 AI coding 视作"智能体整体能力"的试金石。其强调"类人类实习生自主执行多项任务"的定位，意味着大厂正把编码 Agent 从"补完工具"推向"可委托的工程单元"，对 IDE 内置 Agent 的竞争格局形成新压力。

### 2. OpenSquilla 0.4.0 为 AI 编码引入"自我验证"机制
基元律动开源 Agent 项目 OpenSquilla 发布 0.4.0，核心更新是把"红绿回归证据链"内化进编码工作流：先写注定失败的测试给问题定性 → 实现功能使其由红转绿 → 跑原有测试确认未破坏别处；三关全过才交付，否则自动重改到通过（默认修复闭环），且改动只在隔离副本施工、验收合格才落回源码。配合 SquillaRouter 智能路由（按任务复杂度选模型），内测综合成本下降 60–80%。
- **值得关注**：直击 AI coding 当前最大信任瓶颈——"能写不等于能信"。把验证从"人逐行复核"转成"Agent 自证"，是编码 Agent 走向无人值守 / 规模化进入生产环境的关键前提；"隔离施工 + 自愈闭环"思路也值得在本地 3D 工具链的自动化改造中借鉴。

## 二、具身智能

### 3. WAIC 2026 收官：具身智能从"身体竞赛"转向"大脑竞赛"，"一脑多形"成核心关键词
本届具身智能参展企业由去年 80 余家增至 200 余家，集结 208 款终端、超 300 台真机。"能走能跳能翻"的表演叙事退场，**量产 / 适配 / 场景交付** 成为主线。核心变化有三：
- **真实上岗**：60 台机器人于世博/张江/西岸三地四馆部署态公共服务，背后由 GO-2 具身基座大模型 + GE-2 世界模型支撑；
- **一脑多形**：梅卡曼德 Mech-GPT、蚂蚁灵波"通用大脑"均实现同一大模型驱动多构型（人形/工业臂/移动机器人）协同作业，含全球首次人形多机协作；
- **资本转向**：2026 上半年近 440 亿元涌入赛道，其中 55% 流向"大脑"而非硬件形态。
- **值得关注**：行业从"展示可能性"转向"验证可复制性"——比拼点变为"真实产线连续运行 8 小时不出故障"。对 MMD/3D 数字人与物理机器人同源的趋势而言，"一脑多形 + 场景可复制"正是虚拟角色与实体本体可共享同一智能底座的信号。

### 4. 腾讯开源具身基座模型 Hy-Embodied 系列 + Apexio / TairosAgent 智能体
腾讯 RoboticsX 与福田实验室联合混元，系统性打通"感知—身体—行动"闭环并开源：VLM-1.0（"右脑"，以约 1/10 算力逼近上代旗舰）、RxBrain-1.0（"大脑"，统一推理与未来状态想象，建 RxBrain-Bench）、VLA-0.5（"小脑"，RoboDojo 仿真综合第一）；智能体 **Apexio**（认知/感知/执行三层常在线，含生存驱动自保）与 **TairosAgent**（具身原生框架，响应从通用框架几十秒缩至 2–3 秒）。同步升级 Tairos 开放平台（钛螺丝）。
- **值得关注**：这是国内头部厂商首次把"左右脑+大小脑+持续在线智能体"的完整栈开源。其"力觉/触觉/视觉与具身大模型融合"（小六机器人安全接触人）与分层在线架构，对"数字人/虚拟角色如何获得持续感知—决策—行动闭环"有直接参考价值。

### 5. 银河通用 WAM-TTT：全球首个具身"测试时后训练"框架，机器人可"现学现上岗"
银河通用发布 WAM-TTT（World-Action Model Test-Time Training），首次把 TTT 范式从语言模型迁移到物理机器人控制。机器人通过**原始人类实拍视频**在推理阶段完成轻量适配训练：先消化视频场景信息写入临时记忆，再结合既有动作能力执行，既适应新环境又保留原能力，破解"训练精通、部署失灵"的泛化衰减痛点。
- **值得关注**：把 LLM 的"推理时自我提升"路线首次在机器人落地，指向具身智能的"部署后连续学习"时代。对客户而言，换点位不再需重新采数重训——若与腾讯/原力灵机的后训练栈结合，低成本跨场景泛化将显著压低规模化落地门槛。
