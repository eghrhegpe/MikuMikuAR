## 1. OpenAI 把 Codex 引擎「寄生」进 Claude Code：AI 编程竞争从模型军备转向工具链
- **事件**：回溯今年 3/30，OpenAI 为 Claude Code 上线官方插件——开发者一条命令即可让 Claude 负责拆分需求、审核输出，OpenAI 自家 Codex 负责写代码。6 月 Anthropic 发 Fable 5（工程管理升级）+ OpenAI 发 GPT-5.6-Sol（代码精度跃升）后，该插件已演化为「Fable 5 管 / GPT-5.6-Sol 写 / Fable 5 再审」的双模型协同工作流。
- **值得关注**：
  - 模型生成能力正在**商品化**，而「理解工程结构、拆分任务、管理版本、审核变更」的工程管理能力成为稀缺护城河——Claude Code 已是大量开发者默认工作台，不是因为码写得更好，而是因为它「管得住工程」。
  - OpenAI 的博弈：不再逼开发者二选一，转而成为对手工作台里「默认的代码生成引擎」。类比微软 80 年代把 MS-DOS 装进 IBM、为 Mac 做 Office——在别人生态里抢一个不可替代的层。
  - 对 Anthropic 的长期风险：若 Claude Code 核心价值退化为「管理」，动手写的引擎来自 OpenAI，核心智能生成环节话语权可能旁落。
  - **对本项目的启示**：AI coding 的价值锚点正从「模型」转向「工作流编排 + 工具链深度」。WorkBuddy/CodeBuddy 类入口的护城河在协议与集成层，而非单一模型。

## 2. 逐际动力（LimX Dynamics）完成近 2 亿美元 Pre-IPO 轮，Pre-IPO 估值 150 亿元
- **事件**：7/14 深圳通用人形机器人公司逐际动力宣布完成近 2 亿美元 Pre-IPO 轮，半年累计融资 4 亿美元，投后估值 150 亿元。本轮近 70% 资金来自欧洲、中东、北美机构（阿联酋磊石资本、华山资本、GGG Group、Redstone VC 等），IDG、蓝思科技入局，蔚来资本、京东系老股东超额追投。资金重点投向「大小脑融合」算法 + 自研具身 OS，推动数千台全自主人形机器人规模化部署。
- **值得关注**：
  - **资本焦点迁移**：从疯狂追「机器人本体」转向押注「大脑」与关键零部件（灵巧手、关节、电子皮肤、世界模型）。投资人共识——谁能把智能算法嵌入物理世界复杂需求，谁先跑通商业闭环（具身「ChatGPT 时刻」）。
  - **退出端信号**：港股 18C 特专科技板块成为机器人企业首选 IPO 通道，目前港交所秘密递表待审的机器人产业链企业超 40 家。
  - 行业背景：2026 上半年国内具身智能/机器人融资 288 起、披露超 460 亿元，半年融资已破 900 亿（同比约 5 倍）。逐际 LimX Luna（全尺寸交互人形）已交付，过半订单来自海外。

## 3. 启元 T1 发布：全球首个「可变形个人机器人」，WAIC 明日（7/17-20 上海）首秀
- **事件**：上纬新材 7/14 发布消费级具身品牌「启元机器人」新品 T1（Transformer 1），采用行业首创 Transformer 跨形态一体架构，可在**轮足人形 ↔ 四足**形态间自主切换，切入家庭陪伴、户外出行、移动拍摄等场景。将于 WAIC 期间首次公开亮相。
- **值得关注**：
  - 以「形态创新」差异化突围消费级具身，但产品尚未量产——变形架构可靠性、场景刚需与规模化能力仍需市场检验。
  - WAIC 2026 本周开幕，具身 + 智算双赛道各超 200 家企业、300+ 首发，是观察具身商业化窗口的关键节点。

## 4. 1-bit 量化突破：54GB 大模型压至 3.8GB、保留约 90% 能力，浏览器可跑
- **事件**：实测显示 1-bit 量化可将 54GB 大模型压缩至 3.8GB 并保留约 90% 能力，直接在浏览器中运行。同期商汤视觉套件开源、本地化工具链继续降门槛。
- **值得关注**：
  - 大模型本地化从「需一块 4090」降至「随便开个网页」的轻量门槛，端侧 AI（AI PC/手机/机器人本体推理）的基础设施成本骤降。
  - 对端侧 Agent、本地化 AI coding 助手、具身机器人「大脑」本地推理均是降本利好，隐私与离线可用性同步提升。

## 5. AI 编程全面进入 IDE 原生层 + Agent 基础设施升温
- **事件**：
  - **Xcode 27**（WWDC 6/10）原生集成 Anthropic/Google/OpenAI 三大 coding agent，并配本地 Neural Engine 行内补全模型——IDE 正成为多模型 agent 的默认入口。
  - **Codex CLI** 一周内连发 3 版（0.138–0.140）：代码模式内 web 搜索、Claude Code 配置导入（`/import`）、Windows Computer Use、桌面 `/app` 交接、token 追踪。
  - **Oracle** 推出统一 AI Agent 构建工具，企业搭 agent 门槛再降；蚂蚁「阿福」医疗 Agent 工程化落地，腾讯被称「Agent 工厂」。
- **值得关注**：
  - IDE/CLI 正从「编辑器」演变为「agent 编排面」，谁占据开发者日活入口，谁就掌握工作流定义权（呼应第 1 条的工具链之争）。
  - 巨头在 Agent 基础设施层（构建、编排、可观测）的竞争加剧，纯模型 API 的差异化空间被压缩。
