# 编码奇谭：MikuMikuAR 联邦演义

> 在一个 PMX 模型散落于各城邦的世界里，一个桌面壳不发明任何东西，只让所有城邦共享同一个模型库。

---

## 目录结构总览

本小说按 `frontend/src/` 代码目录锚定 10 章 + 1 个附录容器。**改了代码 → 看路径前缀 → 命中对应章 → 更新该章尾部。**

| 章号 | 文件夹 | 对应代码目录 |
|------|--------|------------|
| 01 | `01-基础设施与依赖/` | `frontend/src/core/` |
| 02 | `02-UI交互/` | `frontend/src/menus/` |
| 03 | `03-动作演算/` | `frontend/src/motion-algos/` |
| 04 | `04-音频与换装/` | `frontend/src/outfit/` |
| 05 | `05-布料物理/` | `frontend/src/physics/` |
| 06 | `06-相机移动/` | `frontend/src/scene/camera/` |
| 07 | `07-环境渲染/` | `frontend/src/scene/env/` |
| 08 | `08-模型管理/` | `frontend/src/scene/manager/` |
| 09 | `09-程序化动作/` | `frontend/src/scene/motion/` |
| 10 | `10-灯光与阴影/` | `frontend/src/scene/render/` |

附录 `appendix/` 收纳非代码目录锚定的章节，分 4 组：跨模块重构 / 文档演进 / Go后端 / 安全横切。

判定规则与决策链路详见 [AGENTS.md](AGENTS.md)。
