### 1. reset() 方法 — 初始版本

```typescript
reset(level: PopupLevel): void {
    this.levels = [level];
    this.activeIdx = 0;
    this.buildPanel(this.panels[0], level);
    this.panels[1].style.display = "none";
    this.inner.style.transform = "translateX(0)";
    this.inner.style.transition = "none";
    this.onAfterRender?.(level, this);
}
```

### 2. reset() 断点逐行执行

```
this.levels = [level];          → display 还是 ""
this.activeIdx = 0;             → display 还是 ""

this.panels[1].style.display = "none";   → panels[1] 变 none，panels[0] 还是 ""
this.inner.style.transform = "translateX(0)"; → 还是 ""
this.inner.style.transition = "none";   → 还是 ""
```
