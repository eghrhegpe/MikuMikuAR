模块以 `internal/app` 为核心聚合层，通过 `App` 结构体暴露给 Wails 前端的 ServiceStartup/ServiceShutdown 生命周期；其余子包按职责拆分：
- `app/`：业务入口，`app.go` 定义 App 结构体及配置/最近模型/对话框选择等 API，`library.go` 并行扫描 PMX/VMD/Audio 等资源，`httpserver.go` 实现受信任根 + 路径穿越防护的隔离 HTTP 服务，`watch.go` 基于 fsnotify 做下载目录去抖监听，`plaza_window.go` 预热 WebView2 窗口，`update.go` 检查 GitHub Release。
- 平台抽象采用构建标签 + 同名文件对：`fileaccess.go` + `*_desktop.go`/`*_android.go`、`pathmgr.go` + `*_desktop.go`/`*_android.go`，由 `isAndroid` 变量在运行时切换具体实现。
- `dialogs/file_dialog.go` 封装 Wails v3 Dialog，供 app 统一调用。
- `thumbnail/thumbnail.go` 使用 SHA-256 作为缓存键存取 PNG 缩略图。
- `util/` 纯标准库工具：错误包装 (`WrapError`)、panic 恢复 (`SafeCall`)、PMX 二进制头解析。
依赖方向单向：`app → {dialogs, thumbnail, util}`，`util` 与 `thumbnail` 仅依赖标准库，无循环引用。
