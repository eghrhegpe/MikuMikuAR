## 一、【AI Coding · 头条】工信部提示 Claude Code 存"后门"风险，阿里内部禁用转自研 Qoder

- **事件**：7/13 工信部网络安全威胁和漏洞信息共享平台发布风险提示，指 Anthropic 的 Claude Code 存在安全"后门"隐患，危害严重。7/12 曝阿里已发内部紧急通知，要求员工 7/10 前卸载 Anthropic 全系产品（含 Claude Code 及模型），统一切换自研 Agentic 编程平台 **Qoder**。
- **为何关注**：这是 AI Coding 从"工具选型"上升到**供应链安全与地缘合规**的标志性事件。对国内团队意味着自研/国产 Agentic IDE（Qoder、ZCode、Trae）将获政策性替代红利；对我们这类 TS/Babylon 工程而言，涉及外部 coding agent 的合规审查会成为常态。

## 二、【AI Coding · 竞争格局】GPT-5.6 全家桶落地 + Codex 并入 ChatGPT Work，Meta 首发闭源编程模型 Muse Spark 1.1

- **事件**：OpenAI 正式放出 GPT-5.6（Sol/Terra/Luna），Coding Agent Index 达 80 分刷新 SOTA；同步将 Codex 并入 **ChatGPT Work**，Codex 独立入口消失，走"超级应用"整合路线。同期 **Meta Muse Spark 1.1** 亮相——Meta 首个具竞争力的编程/computer-use 模型，且**闭源**（一改 Llama 开源策略），面向自家 App 生态的自动化任务。
- **为何关注**：编程 Agent 正从"独立 IDE 插件"向"办公超级应用内置"收敛（OpenAI、Anthropic 同向）。Meta 闭源转向意味着可用开源基座进一步收缩，评估 Llama 系工具链的团队需重新排布。

## 三、【AI Coding · 基础设施】智谱首披 GLM-5 系列 Coding Agent 底层突破：吞吐 +132%，异常率降至万三以下

- **事件**：7/13 智谱（02513.HK）首次披露 GLM-5 系列在超大规模 Coding Agent 调用场景的工程优化成果——系统吞吐量最高提升 **132%**，异常输出率从约万分之十降至**万分之三以下**；港股续升逾 7%，里昂上调评级至"跑赢大市"。
- **为何关注**：当前 coding agent 的瓶颈已从"单次代码质量"转向"高并发长链路调用的稳定性与成本"。吞吐/异常率这类工程指标，比单纯 SWE-bench 分数更贴近生产环境（多 AI 协同、长任务）的真实体验。

## 四、【具身智能 · 宏观】上半年融资超 900 亿，投资逻辑从"Demo 叙事"切向"订单与复购"

- **事件**：IT桔子数据显示 2026H1 国内具身智能融资总额超 **900 亿元**（同比增 5 倍），事件超 300 笔（同比 +137%）；6 月单月创新高 63 起、350 亿元。资金高度向头部集中，"大脑派"占比过半，"本体派"仅 12.8%。机构尽调标准从"看论文和演示"转向"看产线和订单复购"（星动纪元千台级交付、部分场景达人工 85% 效率）。
- **为何关注**：赛道进入"兑现元年"前夜（业内预期 2027 为量产兑现元年）。泡沫讨论退居次席，能否拿出出货量与损益表成为分水岭——这为 VLA/世界模型的落地价值提供了更硬的验证锚点。
