# AI 日报 — 2026-07-24

## 1. 【AI coding】OpenAI Codex 连环动作：7/24「codexy」预告 + 上下文窗口悄然缩减

- **事件**：Codex 工程师 Tibo Sottiaux 于 7/23 预告 7/24 将有 Codex 相关发布（"feeling codexy"），社区猜测指向原生 Linux 应用 / 语音控制 / 工作流更新。同期 NetEase 报道，Codex CLI 已将 GPT-5.6 默认输入上下文窗口从 **372K → 272K token**，引发 Reddit/X 对长会话、跨仓库重构的担忧。
- **值得关注**：Codex 在 7/9 刚并入 ChatGPT 桌面端、用户破 800 万，此刻一边放大招一边砍上下文，呈现「能力上探、成本下压」的双重挤压。对联邦的启示：**不要硬编码上下文预算**，长期编码会话需围绕检索 + 模块化设计，避免被单一平台的运行参数绑架（这也是 ADR-105/106 异步生命周期规范的延伸）。

## 2. 【AI coding】DeepSeek 退役 chat/reasoner 命名，V4 Flash/Pro 按 tier+thinking 重构

- **事件**：7/24 15:59 UTC 起，`deepseek-chat` 与 `deepseek-reasoner` 停止响应，统一并入 `deepseek-v4-flash` / `deepseek-v4-pro`。关键陷阱：① `reasoner` 别名实际映射到 **Flash**（非 Pro），静默降级推理质量；② 新版默认开启 `thinking`，纯 rename 会让原低成本调用悄悄变贵。同时上线 Anthropic 格式端点，Claude Code 等可零改路由。
- **值得关注**：这是「命名空间退役 ≠ 模型日落」的典型迁移坑。联邦若在任何 binding/CI 链中硬编码了旧模型名，需在今日完成（tier, thinking）二维路由决策。**值得做成一条工程提醒写进 MEMORY。**

## 3. 【AI coding】腾讯云 CodeBuddy NPC：云端「持证上岗」的自治编码智能体

- **事件**：7/24 发布 CodeBuddy NPC，定位为可在生产级隔离沙箱中执行部署/扩缩容的云端智能体。实测平均排查耗时 117 秒（资深工程师的 1/3.8），支持自动改码、补测试、同步锁文件、重建构建。已落地跨境电商团队（72h 交付 5 模块）与银行信创迁移作战室（Oracle PL/SQL → 达梦 DMQL 转译）。
- **值得关注**：从「写代码」转向「定义问题 + 设定约束 + 校准目标」，开发者角色向「意图架构师」迁移。与联邦的 Agent 化开发方向同频——值得对比其沙箱隔离与权限模型设计。

## 4. 【具身智能】2026 被定为「人形机器人应用元年」：中国整机产量有望破 10 万台

- **事件**：CE 中国经济网 7/24 报道，WAIC 2026 展品已从「能动演示」转向 3C 电子 / 汽车制造 / 仓储 / 家庭等真实场景。数据：中国 AI 核心产业规模破 1.2 万亿元，AI 专利全球占比 60%；2025 年人形机器人出货 1.44 万台（全球 84.7%），2026 年整机产量有望破 10 万；上半年具身领域融资 460 亿元 / 288 起。IDC 四点判断：系统能力竞争、场景规模复制、生态分水岭、中国引领。
- **值得关注**：量产临界点已至，**世界模型**被毕马威/IDC 一致判为破局关键（解决数据稀缺、长链路任务、世界理解→动作转化）。对联邦的 MMD/物理仿真路线，意味着「仿真数据 → 真实闭环」飞轮价值上升。

## 5. 【具身智能】Black Forest Labs 杀入物理 AI + 北京 72 台机器人上岗公园

- **事件**：① 德国 Black Forest Labs（Flux 图像模型出身）正测试首个机器人模型 **Flux 3**，从图像/视频/音频学习并预测下一步动作，已与 Mimic Robotics 合作 Flux-mimic、与奥迪等制造伙伴测试，业务从图像生成扩展至物理 AI。② 北京 14 家市属公园本月底前投用 72 台机器人，覆盖清洁、巡检、导览、水面救援等 6 场景；国产机器人上半年出口暴增 359%。
- **值得关注**：图像生成实验室跨界物理 AI，印证「世界模型 / VLA」正成为跨模态厂商的必争之地；公园规模化部署则标志具身智能从展会走向城市基建级常态化运营。
