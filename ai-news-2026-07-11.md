## 1. Z.ai 发布 ZCode：免费 Agentic IDE，SWE-Bench Pro 反超 GPT-5.5
**事件**：智谱（Z.ai）推出免费编程智能体 ZCode，底座为 GLM-5.2（744B MoE）。SWE-Bench Pro 得分 62.1，超过 GPT-5.5（58.6），逼近 Claude Opus 4.8（66.0）；Terminal-Bench 2.1 得 81.0。定价激进——基础版免费，付费 16.2 美元/月（低于 Cursor Pro 的 20 美元），API 价格约为 Claude Opus 的 1/4，且内置微信/飞书远程控制。
**关注理由**：国产编程 Agent 首次在权威基准上正面压过 OpenAI 主力模型，叠加"免费 + 极低 API 价"，直接冲击 Cursor/Anthropic 的商业化护城河，是中文 AI Coding 市场的关键变量。

## 2. 蚂蚁灵波 LingBot-VA 2.0：提出"具身原生"路线，开源 6B 视频-动作大模型
**事件**：蚂蚁灵波团队发布 LingBot-VA 2.0（arXiv 2607.08639），主张机器人大脑应"具身原生"重构，而非沿用互联网大模型"迁移+微调"范式。核心是语义化视觉-动作 tokenizer + 因果预训练 + MoE + "前瞻推理"，将推理成本压到实时控制阈值以下。已开源代码（Apache-2.0）与 6B checkpoint。
**关注理由**：这是对当前 VLA 主流技术路线的一次系统性质疑与替代方案，且完整开源，为国内具身智能提供了可复现的"世界模型级"底座，值得工程团队跟进评估。

## 3. NVIDIA Isaac GR00T 1.7 接入 Hugging Face LeRobot
**事件**：英伟达将人形机器人 VLA 基础模型 Isaac GR00T 1.7 与人类示教采集框架 Isaac Teleop 直接集成进 Hugging Face 开源库 LeRobot，开发者可用标准化工作流对人形机器人模型做后训练与部署；后续 Cosmos 3 世界模型将补充真实数据稀缺场景下的仿真数据生成。
**关注理由**：具身智能的"数据—训练—部署"链路正在被标准化、开源化，显著降低人形机器人开发门槛，是生态基础设施层面的重要一步。

## 4. 全球首例人形机器人活体手术临床前试验成功
**事件**：加州大学圣地亚哥分校团队用两台 "Surgie" 机器人在活体动物上完成人机协作/机器人团队胆囊切除；另有团队采用宇树 G1 人形机器人手持腹腔镜器械，遥操作为活猪完成标准微创胆囊切除，为全球首例人形机器人在活体中走完完整微创手术流程。
**关注理由**：验证了人形机器人作为低成本、灵活手术助手进入手术室的可行性，是具身智能从工厂/仓储向高精度医疗场景延伸的标志性节点。
