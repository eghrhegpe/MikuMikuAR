## 🧩 AI Coding

### 1. Claude Code 7 月更新：沙箱化内置浏览器 GA、/doctor 交互式修复、智能体成本预算
- **事件内容**：Anthropic 在 v2.1.202–v2.1.207 连续发布四项能力——① 桌面端沙箱化内置浏览器（可自主检索实时文档、校验第三方 API、读取 Figma 设计稿，安全分类器前置审查外部站点操作）；② `/doctor` 升级为交互式诊断/修复器（扫描未用 skill、MCP、重复 CLAUDE.md，并量化上下文窗口成本，确认后改写）；③ 按智能体粒度设成本预算 + checkpointing；④ 7/19 前速率上限 +50%，并延长 Claude Fable 5 访问至 7/19。
- **为什么值得关注**：内置浏览器让 agent 在自主任务链中"自我验证输出 vs 实时文档"，是把 coding agent 从「读代码」推向「对照真实世界校验」的关键一步；`/doctor` 的上下文成本审计思路，直接对应大型项目里 agent 被冗余上下文拖慢的痛点。
- **来源**：subagentic.ai、AGIHunt、QQ 新闻（2026-07-13/14）

### 2. KwaiKAT 发布 KAT-Coder-Pro V2.5：长程工程能力 + AutoBuilder 训练闭环
- **事件内容**：快手 KwaiKAT 团队旗舰 Agentic Coding 模型迭代。核心突破是 **AutoBuilder 自动化环境构建流水线**——将「可运行仓库环境」的构建成功率从业界约 **16.5% 提升至 57.2%**，沉淀覆盖 **12 种编程语言、超 10 万个可运行可验证的真实训练车间**；模型可在几十~上百文件仓库中定位改动、遵循项目规范做最小修改、跑通测试。
- **为什么值得关注**：长程软件工程的真实瓶颈不是「补一段函数」，而是「读懂模糊需求 → 仓库定位 → 最小改动 → 测试验证」的闭环。**可运行环境的自动化构建**（而非静态语料）是 agentic 训练数据稀缺的根因——这条指标直接量化了工程闭环的可达性，对自研 coding agent 的数据管线有强参考性。
- **来源**：163.com / 大京新闻网（2026-07-13）

## 🤖 具身智能 / Physical AI

### 3. 宇树 G1 登《Nature》：全球首例人形机器人活体微创手术
- **事件内容**：基于国产宇树 G1 通用人形机器人，完成全球首例**远程操控活体猪腹腔镜胆囊切除微创手术**。方案依托虚拟 RCM 视觉算法，无需专用手术机械臂，机器人直接握持临床标准腹腔镜器械；单台机器人完整手术耗时 **56 分钟**，双机协同 **32 分钟**。
- **为什么值得关注**：填补了「通用人形机器人 + 精密医疗」场景空白，验证人形硬件在远程基层诊疗/战地医疗/应急手术的可行性。对具身赛道而言，这是从 Demo 级运动控制跨向「高可靠精细操作」的顶刊级背书——也意味着遥操作 + 视觉伺服仍是当前最务实的落地路径（而非纯 RL 自主）。
- **来源**：163.com（前沿在线，2026-07-14）、Nature

### 4. 智源发布 RoboBrain Orca 世界基座模型：12.5 万小时无标注视频训练追平专用系统
- **事件内容**：智源研究院「悟界·RoboBrain Orca」通用世界基座模型，依托 **12.5 万小时海量无标注真实视频**完成预训练，**不依赖机器人动作标签**，下游任务综合性能追平各类专用具身系统。创新「无意识观测学习 / 有意识语义学习」双路径框架，内部构建统一世界表征空间，可灵活切换文本、图像、机器人动作输出头。
- **为什么值得关注**：直接击中行业「机器人训练数据稀缺」痛点——**无标签视频预训练 → 跨场景泛化**的范式，若成立，将大幅降低具身模型对昂贵遥操作/动作标注数据的依赖，是 Physical AI 底层训练路线的轻量化拐点。
- **来源**：163.com（前沿在线，2026-07-14）
