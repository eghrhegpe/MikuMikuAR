## 一、AI Coding

### 1. Kimi K3 登顶 Arena 前端编程榜，开源权重直逼 GPT / Anthropic
月之暗面发布 **2.8 万亿参数** 开源旗舰 Kimi K3（896 专家动态激活 16、100 万 token 上下文、原生视觉理解），上线数小时即在 **Frontend Code Arena 以 1679 分登顶**，前端编程盲评超越 Claude Opus 4.8 与 GPT-5.6 Sol，综合能力稳定压制除 Fable 5 / Sol 外全部模型。完整权重 **7/27** 开放。
- **值得关注**：这是首个跨入 3 万亿量级的开源模型，且首战即在"最硬"的编程任务上打穿闭源护城河。一旦本地可跑，数据敏感 / 高配额团队将获得"自托管可用替代"，AI coding 的模型层选择权正从闭源手中转移。

### 2. Thinking Machines 开源万亿参数多模态模型 Inkling
Inkling 主打 agentic coding 与低 token 消耗，**SWE-bench Pro 62.1、MCP-Atlas 76.8（工具调用）、AIME 2026 达 99.2**；内置 MTP"草稿层"提速，配合 Unsloth 1-bit 量化显存降 95%，可经 llama.cpp / GGUF 在端侧搭建 OpenAI 兼容服务。
- **值得关注**：编程 + 工具调用 + 端侧量化三线拉齐，延续"开源模型下沉到本地 PC 时刻"的趋势，为轻量化 coding agent 提供了新的自托管底座。

## 二、具身智能（WAIC 主场）

### 3. RoboScience「一脑多手」：30 秒换手即用，跨本体灵巧操作破局
自研云端具身大模型 **Visics** 在 WAIC 首秀：多款不同品牌灵巧手搭载同一套模型，实时自主感知、规划并抓取任意物体（含观众随机递物），现场演示 **30 秒热插拔换手、Zero-shot 零样本适配**，商超多 SKU 抓取成功率 >99%。
- **值得关注**：行业长期困于灵巧手"一手一训"，此次首次实现单一具身大模型驱动异构本体，验证了跨本体 / 跨物体 / 跨任务三维泛化——具身"大脑"与"硬件"解耦的关键一步。

### 4. 智元 × 京东物流「精灵 G2 Max」：人形机器人首次仓储生产级落地
双方在 WAIC 发布安规级重载人形机器人 **精灵 G2 Max**，率先部署于京东"智狼仓"，实现人形机器人**首次在真实仓储生产场景的生产级落地**；支持 24h 不间断搬运码垛，前代 G2 已在龙旗工厂质检线连续运行超 100 天、成功率 99.99%。
- **值得关注**：从展台 Demo 转向真实产线"核心作业单元"，是"具身规模化应用元年"最有说服力的商业化坐标；部署周期从 4 个月压缩到 1 周，指向可复制的落地范式。

### 5. 大晓机器人「开悟世界模型 3.1」+ ACE-BRAIN 0.5：世界模型与 VLA 协同路线
商汤系大晓在 WAIC 发布开悟世界模型 3.1（"理解—生成—预测"一体化骨干）与一体化具身基模型 ACE-BRAIN 0.5。董事长王晓刚判断：**世界模型未成熟前将与 VLA 长期协同**，待以人为中心的数据从几十万提升至千万小时量级后，世界模型逐步吸收 VLA，最终收敛为大一统基座。
- **值得关注**：这是对"世界模型 vs VLA"路线之争的一次清醒表态——不是替代而是阶段性并存，并把胜负手明确指向**数据规模**，为判断具身技术栈演进提供了可验证的时间标尺。
